১. প্রোগ্রাম শুরু ও শেষ (Program Structure)

            
যেকোনো কোড লেখার আগে প্রোগ্রামকে বোঝাতে হবে কোথা থেকে কোড শুরু
              এবং কোথায় শেষ।

            
    
              
  • shuru koro : প্রোগ্রামের শুরুতে লিখতে হবে।
    • 
              
  • shesh koro : প্রোগ্রামের শেষে লিখতে হবে।
    • 
            

          


          

            
২. ভেরিয়েবল এবং আউটপুট (Variables & Print)

            
কোনো ডাটা মেমোরিতে ধরে রাখতে ভেরিয়েবল ব্যবহার হয় এবং তা স্ক্রিনে
              দেখাতে প্রিন্ট কমান্ড লাগে।

            
    
              
  • dhoro x = 10; : 'dhoro' কিওয়ার্ড দিয়ে নতুন
                ভেরিয়েবল ডিক্লেয়ার করা হয়।
    • 
              
  • dekhao "Hello"; : টার্মিনালে কোনো টেক্সট বা
                ভেরিয়েবলের মান প্রিন্ট করতে ব্যবহৃত হয়।
    • 
            

          


          

            
৩. কন্ডিশন / শর্ত (If-Else)

            
লজিক চেক করার জন্য কন্ডিশনাল স্টেটমেন্ট ব্যবহার করা হয়।

            
    
              
  • jodi (x > 5) { } : যদি শর্তটি সত্যি হয়, তবে
                ভেতরের কোড কাজ করবে।
    • 
              
  • nahole { } : উপরের শর্তটি যদি মিথ্যা হয়, তবে এটি
                কাজ করবে।
    • 
            

          


          

            
৪. লুপ বা চক্র (Loops)

            
একই কাজ বারবার করার জন্য লুপ ব্যবহার করা হয়।

            
    
              
  • jotokhon (i <= 5) { } : শর্ত সত্যি থাকা পর্যন্ত
                (While loop) কাজ করবে।
    • 
              
  • koro (dhoro i=0; i<5; i=i+1) { } : নির্দিষ্ট
                সংখ্যক বার (For loop) চালানোর জন্য।
